# 🏨 Hôtel Bianiac

Bienvenue sur le projet **Hôtel Bianiac**, un site web moderne et intuitif permettant aux clients de réserver leurs séjours en toute simplicité.

## ✨ Fonctionnalités principales
- 📅 **Système de réservation en ligne**
- 🌐 **Interface utilisateur attrayante et responsive**
- 🏨 **Affichage des chambres et services**
- 📧 **Formulaire de contact** pour toute demande d'information
- 🔒 **Gestion des utilisateurs** (clients et administrateurs)
- 🌟 **Avis et retours clients**

## 📚 Technologies utilisées
- **Frontend :** HTML, CSS, JavaScript (React)
- **Backend :** PHP (Laravel)
- **Base de données :** MySQL
- **Paiement :** Moov, MTN, Celtis, Carte bancaire
- **Notifications :** Email/SMS

## 🛠️ Installation et exécution
1. **Cloner le projet**
   ```sh
   git clone https://github.com/Kcelemmano-26/hotel_bianic.git
   cd hotel_bianic
   ```
2. **Installer les dépendances**
   ```sh
   composer install  # Pour Laravel
   npm install       # Pour React (si applicable)
   ```
3. **Configurer l'environnement**
   - Copier le fichier `.env.example` en `.env`
   - Modifier les variables de connexion à la base de données
   ```sh
   php artisan key:generate
   ```
4. **Lancer le serveur**
   ```sh
   php artisan serve
   ```
   Accéder au site via `http://127.0.0.1:8000`
